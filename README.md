# 🍷 Wine Quality Prediction Web App

A Machine Learning web application that predicts the quality of wine based on its physicochemical properties.

This project combines Machine Learning with Flask web development to create a real-time prediction system where users can input wine features and instantly receive quality predictions.

---

# 🚀 Features

- Predict wine quality using Machine Learning
- Interactive web interface using Flask
- Real-time prediction system
- User-friendly HTML/CSS frontend
- Integrated ML model with backend
- Deployment-ready project structure

---

# 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- XGBoost
- NumPy
- Joblib
- HTML
- CSS

---

# 📊 Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Model Evaluation
6. Model Saving using Joblib
7. Flask Integration
8. Frontend Development
9. Deployment
